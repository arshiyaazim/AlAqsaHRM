Import Employee Data from Excel File
Instructions:

Upload an Excel file (.xlsx, .xls) containing employee data with the following columns:

  • Column A → "Employee ID" (Required)
  • Column B → "Name" (Optional)
  • Column C → "Daily Wage" (Optional)
  • Column D → "Mobile Number" (Optional)
  • Column E → "NID / Passport Number" (Optional)
  • Column F → "Designation" (Optional)
  • Column H → "Join Date" (Optional)

Note: Only rows with a non-empty Employee ID (Column A) will be processed. All other columns are optional.
