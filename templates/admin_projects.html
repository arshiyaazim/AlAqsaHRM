About Projects
What are projects?

Projects help you organize attendance records by different work sites, teams, or assignments. Each project can have its own custom fields for additional data collection needs.

Custom Fields

Custom fields allow you to collect specific data for each project. You can add fields like "Supervisor", "Task Type", or any other information you need to track.
