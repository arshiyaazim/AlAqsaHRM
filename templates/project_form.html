Help & Tips
About Projects

Projects help organize attendance records by site or work locations. Each attendance record can be associated with a specific project.

Custom Fields

Custom fields allow you to add additional information to projects. Examples include:

  • Client name
  • Project manager
  • Budget code
  • And more!

Custom fields can be added using the JSON editor or with the more user-friendly Custom Fields Manager after creating the project.

Tip: Active projects appear in the employee clock-in/out form, while inactive ones only show in reports.
